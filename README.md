## Initialize
### Backend
```bash
    cd backend
    python -m venv venv
    ./venv/Scripts/activate
    pip install -r requirements.txt
```

### Frontend
```bash
    cd frontend
    npm install
```

## Run

Separated terminals

### Frontend Server
```bash
    cd frontend
    npm start
```
### Backend Server
```bash
    cd backend
    flask run
